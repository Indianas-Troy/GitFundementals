# git config

When first setting up git, youll need to work with the git config command in order to set up ypur identity.

For Example, your identity can be set up with the commands such as:
```
git config --global user.name "Arnold Shortman"
git config --global user.email arnoldshortman202@gmail.com
```

For more information, view the [first time git setup](https://git-sm.com/book/en/v2/Getting-Started-First-Time-Git-Setup) documentation.
